# Toram Online - Trait Database

A comprehensive and elegant web application for exploring Toram Online's Drop Item Special Abilities system. Built with vanilla JavaScript and Three.js for an immersive visual experience.

## Features

### Complete Database
- **200+ Traits** across 5 tiers (I-V)
- **Tier Transfer System** mechanics and information
- **12 Game Mechanics** with detailed explanations
- Real-time search and filtering capabilities

### Professional Design
- **3D Particle System** powered by Three.js (1,500+ animated particles)
- **Custom SVG Icons** - Fully professional game-themed graphics
- **Glassmorphism UI** with backdrop blur effects
- **Animated Tier Icons** with unique effects for each tier
- **Responsive Design** optimized for all screen sizes

### Advanced Filtering
- Filter by **Tier** (I, II, III, IV, V, Transfer Info)
- Filter by **Category** (Stats, Combat, Speed, Regen, Proc, Debuff, Utility)
- **Real-time search** across all trait names and descriptions
- **Live statistics** showing filtered results

### Trait Categories
- **Combat** - ATK, MATK, Critical Rate, Accuracy, Defense
- **Stats** - STR, INT, VIT, AGI, DEX, HP, MP
- **Speed** - Attack Speed (ASPD), Cast Speed (CSPD)
- **Regen** - HP/MP Regeneration
- **Proc** - Special activation abilities
- **Utility** - EXP Gain bonuses
- **Debuff** - Status effect limitations

## Tech Stack

- **HTML5** - Semantic markup structure
- **CSS3** - Advanced animations, gradients, glassmorphism effects
- **JavaScript (ES6+)** - Pure vanilla JavaScript
- **Three.js** - 3D graphics and particle system
- **No build tools required** - Direct browser execution
